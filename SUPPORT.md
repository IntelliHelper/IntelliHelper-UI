## Bugs and features

- **Bugs / features:** [GitHub Issues](https://github.com/IntelliHelper/IntelliHelper-UI/issues)  
  Please use the issue templates and include a minimal reproduction when possible.
- **Pull requests:** see [CONTRIBUTING.md](./CONTRIBUTING.md)

## Security

Report vulnerabilities privately — see [SECURITY.md](./SECURITY.md).  
Do not post exploit details in public issues.

## Community standards

Participation is governed by our [Code of Conduct](./CODE_OF_CONDUCT.md).

## What we cannot provide

This is an open-source project maintained by volunteers and the IntelliHelper team.
We cannot guarantee SLA-backed commercial support via GitHub issues. For product
or partnership questions, use the channels listed on
[github.com/IntelliHelper](https://github.com/IntelliHelper).
